This WordPress plugin displays an estimated reading time above the content on single blog posts.

## Features
- Calculates word count
- Estimates reading time (based on 200 WPM)
- Automatically shows it on blog posts

## Installation
1. Copy to `/wp-content/plugins/simple-reading-time`
2. Activate in WordPress > Plugins
3. Done!
